## Leer contenido json desde una web

``` Powershell

$routes = @{
    "/" = { return '{
  "alumnos": {
    "alumno": [
      {
        "operacion": "1",
        "usuario": "juanito"

      {
        "operacion": "2",
        "usuario": "luis"
      }
    ]
  }
}' }
}
``` 
